WooCommerce RESTful service

README :

Welcome to WooCommerce RESTful service. This webservice is build to return custommers and newsletter subscribers of WooCommerce MySQL database for the NABALSY company.

Documentation API :

REST client

Methods :

  • @GET ( /status: String) ................................... Operating test method for the web service.
  • @GET ( /version: String) ................................. Method that returns the version of the web service.
  • @GET ( /wc-users: Object) ............................. Method that returns all customer information from a WooCommerce database in a structured way, customer by customer (XML or JSON).
  • @GET ( /wc-users/{id}: Object) ...................... Method that returns all customer information from a WooCommerce database in a structured way, for a specific id (XML or JSON).
  • @GET ( /wc-users/{from}/{to}: Object) ......... Method that returns all customer information from a WooCommerce database in a structured way, from an id and to a number of results (XML or JSON).
  • @GET ( /wc-users/count: int) ......................... Method that returns the number of customers registred in a WooCommerce database.
  • @GET ( /wysija-users: Object) ....................... Method that returns all member information from a Wysija database in a structured way, customer by customer (XML or JSON))
  • @GET ( /wysija-users/{id}: Object) ................ Method that returns all customer information from a Wysija database in a structured way, for a specific id (XML or JSON).
  • @GET ( /wysija-users/{from}/{to}: Object) ... Method that returns all customer information from a Wysija database in a structured way, from an id and to a number of results (XML or JSON).
  • @GET ( /wysija-users/count: int) ................... Method that returns the number of customers registred in a Wysija database.

Informations :

  • @GET index.html ............................................ Home Page of this web service.
  • @GET readme.html ........................................ Basic information page of this web service.
  • @GET documentation.html ........................... Documentation of this web service.
